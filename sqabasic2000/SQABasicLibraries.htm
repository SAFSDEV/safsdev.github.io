SQABasic Libraries

(Last Updated: Sep.10.2003)

The following lists the libraries currently under development with various versions of Rational's TeamTest product (Rational Robot). Robot V2000 and beyond can use these libraries without issue. The libraries must be compiled with the version of Robot that will run them.

These are generally application independent libraries that are useful in an overall scheme of automated test with SQA Robot.ï¿½ They are constantly being enhanced and refined and new libraries appear fairly often.

General Libraries, DD Utility Libraries, DDEngine Libraries, Component Functions

  • CWPHooker DLL Library for Popup Menus.
  • DatabaseUtilities for accessing ODBC databases.
  • ExcelUtilities Excel Automation Server routines.
  • FileInfoFIFO A custom Buffer type using BufferUtilities.
  • FileTestUtilities: For testing files and directories (needs work).
  • FileUtilities: General Purpose File Utilities.
  • ImageUtilities: Robot Image Capture and Compare.
  • LogUtilities Implements multipoint logging.
  • MenuUtilities for working with Menus and Popups.
  • NTCommandUtilities use some DOS-based commands.
  • ProcessContainer to See GUI Children and Properties

  • STAFUtilities STAF access functions. STAF Home Page

  • WIN32 API Header info for using the Win32 API.
