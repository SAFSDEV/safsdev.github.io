      • install

        public boolean install(java.lang.String... args)
        Currently assumes STAF has been installed and STAFDIR is already set. Can set STAFDIR if args[0] is given a valid value to use. If args[0] is provided, the path will be validated before it is used. Adds STAF paths to CLASSPATH and PATH Environment Variables as long as JSTAF.JAR is found to exist.

      • uninstall

        public boolean uninstall(java.lang.String... args)
        Unset the System Environment Variables used by SAFS for STAF. Remove STAFDIR and paths from CLASSPATH and PATH Environment Variables. Can unset STAFDIR if args[0] is given a value to use.
        Parameters:
        args -
        Returns:

      • main

        public static void main(java.lang.String[] args)
        Main Java executable. Primarily to run standalone outside of a larger process.

        %SAFSDIR%\jre\bin\java org.safs.install.STAFInstaller C:\STAF

        %SAFSDIR%\jre\bin\java org.safs.install.STAFInstaller -u C:\STAF

        System.exit(0) on perceived success.
        System.exit(-1) on perceived failure.

        Parameters:
        args - -- -u to perform an uninstall instead of install.
        Any first arg other than (-u) will be considered the path for STAFDIR.
        The path for STAFDIR does NOT need to be provided if the System Environment Variable already exists with a valid setting.

        SilentInstaller
