      • Select

        public static boolean Select(Component combobox,
             java.lang.String itemtext,
             java.lang.String... extraParams)
        Select an item in Combo Box. Typically a verification that the item was selected is attempted.

        See Detailed Reference

        Parameters:
        combobox - Component (from App Map) to Select item from.
        itemtext - -- The combo box option to select.
        extraParams - optional
          extraParams[0] forceRefresh String, determine if force refreshing after selection. It is usually used when the 'id' of ComboBox is dynamic.
          true, force refreshing
          false, not force refreshing
        Returns:
        true if successful, false otherwise.
        See Also:
        SAFSPlus.prevResults, TestRecordData.getStatusCode(), TestRecordData.getStatusInfo()
        Example:
         boolean success = ComboBox.Select(Map.Google.Combobox1, "ItemText");
 boolean success = ComboBox.Select(Map.Google.Combobox1, "ItemText", "true"); // force refreshing when dealing dynamic 'id' ComboBox
