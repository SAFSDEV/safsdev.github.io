      • AutoItRs.Window

        public AutoItRs.Window(java.lang.String rs)
        The "window text" consists of all the text that AutoIt can "see". This will usually be things like the contents of edit but will also include other text like:
        1. Button text like &Yes, &No, &Next (the & indicates an underlined letter)
        2. Dialog text like "Are you sure you wish to continue?"
        3. Control text
        4. Misc text - sometimes you don't know what it is :)
        The important thing is that you can use the text along with the title to uniquely identify a window to work with. When you specify the text parameter in a window function it is treated as a substring.
