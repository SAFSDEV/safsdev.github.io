      • compile

        public static void compile(java.lang.String classnames)
                    throws SAFSException
        Compile the java/groovy source code.
        NOTE:
        1. It is supposed the "source codes" resides in the "src" sub-folder of the root directory.
        The "compiled classes" will be put the the "bin" sub-folder of the root directory.
        2. Any dependency jars should be included in the environment "CLASSPATH" before this compilation.
        File structure:
         root
   + "src" (source files folder)
      + package1
      + package2
   + "bin" (compiled classes go into this folder)
        Parameters:
        classnames - String, the class names to compile separated by space, such as my.package.ClassA my.pack2.ClassB
        this parameter could be mixed with java and groovy class.
        Note: For groovy, we have to specify all dependency classes in order.
        Throws:
        SAFSException - if the compilation failed.
