      • compile

        public static void compile(java.lang.String classnames)
                    throws SAFSException
        Compile the java/groovy source code.
        NOTE:
        1. It is supposed the "source codes" resides in the "src" sub-folder of the root directory.
        The "compiled classes" will be put the the "bin" sub-folder of the root directory.
        2. Any dependency jars should be included in the environment "CLASSPATH" before this compilation.
        File structure:
         root
   + "src" (source files folder)
      + package1
      + package2
   + "bin" (compiled classes go into this folder--which will be created if not already present.)
        Parameters:
        classnames - String, the class names to compile separated by space, such as my.package.ClassA my.pack2.ClassB
        this parameter could be mixed with java and groovy class.
        Note: For groovy, we have to specify all dependency classes in order.
        Throws:
        SAFSException - if the compilation failed for any reason--including bad parameter values.

      • compile

        public static void compile(java.lang.String classnames,
           java.lang.String sourceDIR,
           java.lang.String outDIR,
           java.lang.String useClasspath)
                    throws SAFSException
        Compile java and/or groovy source code.
        Parameters:
        classnames - String, the class names to compile separated by space, such as my.package.ClassA my.pack2.ClassB
        this parameter could be mixed with java and groovy class.
        Note: For groovy, we have to specify all dependency classes in order.
        sourceDIR - String, the root directory off which the sourcecode files reside.
        outDIR - String, the root directory off which the compiled classes will be stored. The directory will be created, if necessary.
        useClasspath - String, the semi-colon separated CLASSPATH to be used for the compile.
        Throws:
        SAFSException - if the compilation failed for any reason--including bad parameter values.
