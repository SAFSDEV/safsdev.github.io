        public java.util.Map<java.lang.String,java.lang.String> getPersitableFields()
        Description copied from class: PersistableDefault
        This default implementation get all the declared fields Class.getDeclaredFields() and put their name as 'fieldName' and 'persistKey' into the 'persistableFields Map'. But if PersistableDefault.ignoreFieldForPersist(Field) returns true, then that field will not get into 'persistableFields Map'.
