      • appendHeaderValue

        public static java.lang.String appendHeaderValue(java.lang.String headers,
                                 java.lang.String headerName,
                                 java.lang.String headerValue)
        Given a multi-line header String, append a header value to an existing header name, or create the new header with the given value.

        Example: appendHeaderValue(headers, "Accept", "text/html");

        This does NOT act on prestored headers, only the header String provided.

        When appending, this routine will prefix the value with a separator.

        Parameters:
        headers - multi-line header string.
        headerName - name of the header to be created or appended.
        headerValue - value to add to the provided header name.
        Returns:
        multiline headers properly appended, or unmodified if invalid arguments are provided.

      • removeHeaderValue

        public static java.lang.String removeHeaderValue(java.lang.String headers,
                                 java.lang.String headerName,
                                 java.lang.String headerValue)
        Given a multi-line header String, remove a header value from an existing header name, or create the new header without the given value.

        Example: removeHeaderValue(headers, "Accept", "text/html");

        This does NOT act on prestored headers, only the header String provided.

        When removing, this routine will attempt to fix any separators.

        Parameters:
        headers - multi-line header string.
        headerName - name of the header to have a value removed.
        headerValue - value to remove.
        Returns:
        multiline headers properly changed, or unmodified if invalid arguments are provided.
