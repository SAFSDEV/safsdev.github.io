    public class Component
extends DefaultRefreshable
implements IWebAccessibleInternetRole, IOperable
    A library class to handle generic functionalities, such as Click, HoverMouse, GetGUIImage etc. for all kinds of component.

      • inputComponentBoxChars

        public void inputComponentBoxChars(java.lang.String libName,
                          java.lang.String text)
                            throws SeleniumPlusException
        Purpose: Set the text as the content of Component Box.
        This method will not deal with special keys as + --> ShiftKey ^ --> CtrlKey.
        All text will be treated as literal characters.
        For example: the special key "^(v)" will just be treated as literal "^(v)" without any interpretations.
        Parameters:
        libName - String, the concrete Component name of class, which calls 'inputComponentBoxChars()' method, like 'EditBox', 'ComboBox'.
        text - String, the content to be entered into Component box.
        Throws:
        SeleniumPlusException

      • inputComponentBoxKeys

        public void inputComponentBoxKeys(java.lang.String libName,
                         java.lang.String text)
                           throws SeleniumPlusException
        Purpose: Set the text as the content of Component Box.
        This method will deal with special keys.
