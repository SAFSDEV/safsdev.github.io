        public static boolean enableClickListenerFailures
        Default true. Set to true if we want an Exception to be thrown when we perform a Click and attempt to verify through event listeners that the Click actually happened. The problem is, that so many Click verifications fail due to the page and or WebElement going stale or being overwritten that Click verifications issue failures even though the Click was successful.
