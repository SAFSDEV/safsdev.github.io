      protected boolean isShowOnPage(Element element)
      For menu, the menuitem is usually NOT shown on page.
      The method EmbeddedObject.isShowOnPage(Element) in super-class will
      try to search the related web-element (which doesn't exist yet) and will waste
      a lot of time.
      So override it, and just return false, so that we will always call showOnPage(Element)
      to make it visible on page and can be selected correctly.
