      • patternMatch

        public static boolean patternMatch(java.lang.String text,
                   java.lang.String pattern)
        Attempt a Selenium 1.0 pattern match on a given text string.

        Selenium 1.0 patterns are:

        • glob:pattern, supports "*" and "?".
        • regexp:regexp, matches based on the regular expression.
        • regexpi:regexpi, matches based on case-insensitive regular expression.
        • exact:string, an exact verbatim match.
        If no pattern is specified, then 'glob' is assumed to be default.

        Parameters:
        text - String to compare against the pattern
        pattern - String that may include the pattern prefix for the type of comparison to be performed. If no pattern prefix is present, then 'glob' is assumed.
        Returns:
        true if the text matches the provided pattern. false otherwise. if both parameters are null this is considered a match.
