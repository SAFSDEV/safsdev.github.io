Class CreatePathCallback

  • java.lang.Object
    • org.safs.projects.common.projects.callbacks.CreatePathCallback

  • public abstract class CreatePathCallback
extends java.lang.Object
    This callback is expected to be used by projects such as SeleniumPlus. The idea is that SeleniumPlus code will create this callback. Then, at a later time, Core code will call createPath. The SeleniumPlus code that implements createPath will call the Eclipse API to create the path.
