org.safs.projects.common.projects.pojo

Class POJOProject

  • java.lang.Object
    • org.safs.projects.common.projects.pojo.POJOProject

  • public class POJOProject
extends java.lang.Object
    This is a POJO implementation if a Project similar to the Eclipse IProject. This implementation will be used by a project that does not use something like Eclipse. Those projects should call the init() method. For projects that use something like Eclipse, they will likely use a subclass that will hold an Eclipse IProject and will delegate calls to it. They are not required to call the init() method.
