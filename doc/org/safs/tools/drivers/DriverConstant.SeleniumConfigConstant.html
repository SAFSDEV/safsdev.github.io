      • SELENIUMSERVER_JVM_OPTIONS

        public static final java.lang.String SELENIUMSERVER_JVM_OPTIONS
        "SELENIUMSERVER_JVM_OPTIONS" define the JVM Options for SELENIUM server. SELENIUMSERVER_JVM_Xms and SELENIUMSERVER_JVM_Xmx are finer, if they exist then they will be used instead of the options specified in SELENIUMSERVER_JVM_OPTIONS.
        For example, the we have specified option as following, then 4g and 512m will be used to start the selenium server.
        • SELENIUMSERVER_JVM_OPTIONS="-Xms256m -Xmx1g"
        • SELENIUMSERVER_JVM_Xmx=4g
        • SELENIUMSERVER_JVM_Xms=512m
        To use a very high memory, 4g for example, you may need to specify the SELENIUMSERVER_JVM to use 64 bits Java.
        See Also:
        SELENIUMSERVER_JVM_Xms, SELENIUMSERVER_JVM_Xmx, Constant Field Values
