Class EmbeddedInputService

  • java.lang.Object
    • org.safs.staf.service.input.AbstractSAFSInputService
      • org.safs.staf.service.input.EmbeddedInputService
  • All Implemented Interfaces:
    ServiceInterface

    public class EmbeddedInputService
extends AbstractSAFSInputService
implements ServiceInterface
    A subclass of SAFSINPUT service intending to run embedded in the same JVM/Process as the Driver using it. That is, it does NOT run as a STAF service in a separate JVM. However, it can interface to STAF as needed.

    For this to work, the InputInterface classes must be properly initialized to know to use this SAFSINPUT service instead of the standard STAF SAFSINPUT service.
