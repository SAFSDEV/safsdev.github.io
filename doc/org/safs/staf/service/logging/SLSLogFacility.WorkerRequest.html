Class SLSLogFacility.WorkerRequest

  • java.lang.Object
    • org.safs.staf.service.logging.SLSLogFacility.WorkerRequest
  • Enclosing class:
    SLSLogFacility

    protected class SLSLogFacility.WorkerRequest
extends java.lang.Object
    This class is the queued request for the worker thread of this log facility.

    A request contains a command identifier, which tells the worker thread what action to perform, and message specifications if the command is to log a message.
