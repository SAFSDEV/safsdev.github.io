      • parseFiledValue

        protected java.lang.String parseFiledValue(java.lang.Object value)
        Parse the parameter value and convert it to a string which will be persisted to (File or Variable or ...)
        The following are the ways to parse value:
        • The String value will be PersistorToFile.escape(String) and double-quoted if needed.
        • The array value will be transformed into [item1, item2, item3]
        • The Number value will be returned as string without quote.
        Parameters:
        value - Object, the filed's value
        Returns:
        String, the parsed result of the field's value
        null if the parameter 'value' is null.
        Throws:
        SAFSException
