      • autorun

        public void autorun(java.lang.String[] args)
             throws java.lang.Throwable
        This is the critical method users could call to commence the automatic instantiation, configuration, and execution of JSAFSTest methods.

        Minimalist example: 

               public static void main(String[] args)throws Throwable{
         MyTestApp app = new MyTestApp();
         
         new Runner().autorun(args);
         ...
     }

        Automatic configuration and usage is not required. The user can control test configuration and execution within their custom code if they want.

        Parameters:
        args - passed in from command-line Java-- the primordial main(String[] args)
        Usually, it is the class who calls Runner.autorun() will be considered as
        the class that is requesting the automatic configuration and execution, but we can also
        change it by providing parameter as "-autorunclass autorun.full.classname".
        Throws:
        java.lang.Throwable
        See Also:
        JSAFSTest
