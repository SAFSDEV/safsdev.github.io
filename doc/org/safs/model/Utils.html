      • concat

        public static java.lang.String concat(java.lang.String string1,
                      java.lang.String string2)
        Returns a SAFS concatentation of the specified strings using the concat operator ('&').

        For example, the result of calling concat with "^newName" and "^currentDate" is "^newName&^currentDate".

        Parameters:
        string1 - the first string to concatentate
        string2 - the second string to concatenate
        Returns:
        string1 and string2 in a SAFS concatentation.
