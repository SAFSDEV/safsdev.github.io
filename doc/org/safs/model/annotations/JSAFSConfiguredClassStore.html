org.safs.model.annotations

Interface JSAFSConfiguredClassStore

  • All Known Implementing Classes:
    AbstractRunner, DefaultRunner, EmbeddedHookDriverRunner, Runner

    public interface JSAFSConfiguredClassStore
    Required interface for a class that intends to store the object instances that will be automatically created by the automatic dependency injection and execution processes in the Utilities Class.

    This allows examination and use of these objects after the automated processing has completed.

    Author:
    Carl Nagle OCT 15, 2013
    See Also:
    Runner, Utilities.autoConfigure(String, JSAFSConfiguredClassStore)
